== Description ==

BKiAI Chat Free adds a configurable AI chat to your WordPress website and is designed as the public free edition of the BKiAI Chat product line.

This version focuses on a clean Free feature set while transparently showing which capabilities belong to the higher editions. This gives users a clear starting point and a straightforward upgrade path without confusion.

= Included in BKiAI Chat Free =

* AI chat integration for WordPress
* Voice recording in the Free version
* Chat border customisation
* Border thickness setting
* Border colour setting
* One active bot in the Free version
* One uploaded knowledge file for Bot 1
* Clear compare and upgrade area inside the plugin
* Transparent overview of Free, Pro, and Expert differences
* GPT model comparison overview

= Product line =

* BKiAI Chat Free
* BKiAI Chat Pro
* BKiAI Chat Expert

BKiAI Chat Free only contains the intended Free feature scope. Higher-plan capabilities remain visible in the interface where appropriate so users can understand the differences between editions.

= Why use BKiAI Chat Free? =

* Clear entry version for WordPress users
* Transparent distinction between Free and higher plans
* Voice input available in the Free edition
* Practical chat styling options
* Structured comparison inside the plugin

= Free vs Pro and Expert =

Some advanced capabilities are intentionally reserved for licensed Pro and Expert plans. In the Free version, those options may remain visible for transparency, but they are not available for use until the relevant plan is activated.

= External services and privacy =

BKiAI Chat Free sends chat requests to OpenAI in order to generate chatbot responses. Depending on how the site owner configures the plugin, user prompts, system prompts, and optional uploaded knowledge-file content may be transmitted to OpenAI for response generation.

Site owners are responsible for reviewing their privacy notice, data-processing obligations, and the terms of the external services they choose to use.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/bkiai-chat-free/` directory, or install the plugin through the WordPress plugins screen.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Open the BKiAI Chat settings in the WordPress admin area.
4. Configure the general settings and Bot 1 settings as needed.
5. Embed the chat window on your website using the shortcode `[bkiai_chat bot="1"]`.
6. If you use the popup chat mode, configure it through the settings in Bot 1.
7. Save your settings and test the chat on the frontend.

== Frequently Asked Questions ==

= What is BKiAI Chat Free? =

BKiAI Chat Free is the free edition of the BKiAI Chat product line for WordPress. It provides the intended Free feature set and includes a transparent comparison with the Pro and Expert editions.

= What is included in the Free version? =

The Free version includes the free plugin functionality, voice recording, chat border customisation, border thickness setting, border colour setting, one active bot, one uploaded knowledge file for Bot 1, and a built-in compare and upgrade area.

= Is voice recording available in the Free version? =

Yes. Voice recording is enabled in BKiAI Chat Free.

= Can I customise the chat border? =

Yes. The Free version includes chat border settings, including border thickness and border colour.

= How do I embed the chatbot on my website? =

Use the shortcode `[bkiai_chat bot="1"]` to embed the Free bot on a page or post.

= Can I use Bot 2 to Bot 20 in the Free version? =

No. The Free version is limited to one active bot. Additional bot slots belong to the higher plans.

= Are image generation and PDF generation included in the Free version? =

No. These features belong to the licensed Pro and Expert plans. They may remain visible in the interface for transparency, but they are not available for use in the Free version.

= Does the plugin use external services? =

Yes. The plugin sends chat requests to OpenAI in order to generate responses.

= Is BKiAI Chat Free suitable as a starting version? =

Yes. It is designed as a clear entry version that lets users work with the Free feature set and understand the upgrade path easily.
